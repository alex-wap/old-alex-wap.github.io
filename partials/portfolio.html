Alex Wapniarski's Portfolio


Typing Dojo

Typing Dojo is a typing game that lets you race against other users in real time using web sockets. Race your friend or challenge yourself to type as fast as you can!

Technologies Used:
JavaScript, MongoDB, Express.js, AngularJS, Node.js, Socket.io, Materialize, AWS EC2, Git, HTML, CSS.


Crime Dodger

Crime Dodger helps users travel safely through the city of San Francisco. This app maps the crime in San Francisco using Socrata Open Data and Google Maps JavaScript API.

Technologies Used:
JavaScript, Python, HTML, CSS, Bootstrap, MVC, S3, AWS, Git, Socrata Open Data, Google Maps JavaScript API.
