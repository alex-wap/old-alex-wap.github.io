Typing Dojo

Typing Dojo is a typing game that lets you race against other users in real time using web sockets.

Crime Dodger

Crime Dodger helps users travel safely through the city of San Francisco. This app maps the crime in San Francisco using Socrata Open Data and Google Maps JavaScript API.

Technology Used:
JavaScript, Python, HTML, CSS, Bootstrap, MVC, S3, AWS, GitHub, Socrata Open Data, Google Maps JavaScript API.
